# INNOVY - Consultoria Especializada em ERP Totvs

Site institucional da INNOVY, consultoria especializada em ERP Totvs para empresas sem TI interna.

## 🎯 Sobre o Projeto

A INNOVY é uma consultoria que atua como braço de TI externo para empresas que não possuem equipe interna especializada em sistemas de gestão. Somos especialistas certificados em **ERP Totvs**, oferecendo soluções completas em:

- ✅ Implantação de ERP Totvs
- ✅ Consultoria em processos empresariais
- ✅ Suporte técnico especializado
- ✅ Integrações e automações
- ✅ Relatórios gerenciais (Excel/Power BI)
- ✅ Soluções em nuvem e infraestrutura
- ✅ Coordenação e gestão de TI

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido com as seguintes tecnologias:

- **React 18** - Biblioteca JavaScript para interfaces
- **TypeScript** - Tipagem estática para JavaScript
- **Vite** - Build tool moderna e rápida
- **Tailwind CSS** - Framework CSS utility-first
- **shadcn/ui** - Componentes de UI acessíveis e customizáveis
- **Lucide React** - Ícones modernos
- **React Router DOM** - Roteamento

## 📋 Pré-requisitos

- Node.js (versão 16 ou superior)
- npm ou yarn

## 🔧 Instalação

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/innovy-site.git

# Entre na pasta do projeto
cd innovy-site

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

O site estará disponível em `http://localhost:5173`

## 📦 Build para Produção

```bash
# Gerar build de produção
npm run build

# Preview do build
npm run preview
```

## 🌐 Deploy

O projeto pode ser facilmente deployado em plataformas como:

- **Vercel** (recomendado)
- **Netlify**
- **GitHub Pages**
- **Lovable** (deploy com um clique)

### Deploy na Vercel

```bash
npm install -g vercel
vercel --prod
```
